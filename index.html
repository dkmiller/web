About me

I'm a fourth-year PhD student working under Ravi Ramakrishna. Mathematically, I work in algebraic number theory and arithmetic statistics. More specifically, take an elliptic curve E/Q. I'm interested in the interaction between the Galois representation arising from E (in particular, its trace, which counts the number of Fp-valued points of E) and the statistics of symmetric powers of its Satake parameters. I am also interested in time and location-centric applications of Cloud Computing.

It is an unfortunate fact that the copies of SGA in most university libraries are the old Springer versions, which are not exactly paragons of typesetting. Fortunately, the first four volumes of SGA have been typeset using LaTeX. To the best of my knowledge, only the first volume is widely known about, and the typeset version of SGA 4 is difficult to find. All parts of SGA included here have had a digital table of contents added via JPdfBookmarks. Hopefully it will be useful to the mathematical community to have all typeset volumes of SGA in one place. Important note: I didn't create any of these TeX versions except SGA 4.5.

  • SGA 1, and its page on the arXiv
  • SGA 2, and its page on the arXiv
  • SGA 3, volume 1, volume 2, and volume 3. These can be found at Patrick Polo's webpage, but only in individual sections. I used PDFtk to compile the separate sections into a single pdf.
  • SGA 4, volume 1, volume 2, and volume 3. I found them here. (Note: even though volume 3 is titled "tome 1," it actually is volume 3. While the source code can be found e.g. here, I haven't tried recompiling it with the correct title. It looks Yves Laszlo is working on SGA 4, c.f. this page.
  • SGA 4½. While the initial typesetting process is complete, there are still many typos. Feedback is welcome, and if you would like to help with the editing process, its GitHub page is here.
